SADP API Documentation

RESTful API for Healthcare AI Agent Platform

Version: 1.0.0 Base URL: https://sadp-agent-runtime-355881591332.us-central1.run.app Format: JSON

üöÄ Quick Start

1. Test the API

# Check if API is running
curl https://sadp-agent-runtime-355881591332.us-central1.run.app/health

# Get list of available agents
curl https://sadp-agent-runtime-355881591332.us-central1.run.app/agents/supported

2. Execute an Agent

# Execute clinical agent
curl -X POST https://sadp-agent-runtime-355881591332.us-central1.run.app/agents/clinical/execute \
  -H "Content-Type: application/json" \
  -d '{"data": {"input": "Patient data here"}}'

üîê Authentication

Currently: No Authentication Required (Demo Mode)

Production Authentication (Coming Soon)

In production, the API will use JWT Bearer tokens:

curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  https://api.sadp.health/agents/clinical/execute

Note: For integration testing, request a demo API key by contacting support@sarthi.health

POST /agents/{agent_type}/execute Most Important

Execute a specific AI agent with input data

Path Parameters

Parameter Type Description
agent_type string Type of agent (e.g., "clinical", "billing")

Request Body

{
  "request_id": "req_123456",  // Optional
  "data": {
    "input": "Patient: John Doe, Age: 45...",
    "format": "text",  // text, json, hl7
    "context": {
      "user_id": "usr_789",
      "session_id": "sess_456",
      "facility": "General Hospital"
    }
  },
  "metadata": {
    "priority": "high",
    "audit": true,
    "compliance_check": true
  },
  "options": {
    "timeout": 30000,  // milliseconds
    "confidence_threshold": 0.85
  }
}

Response Example

{
  "request_id": "req_123456",
  "agent_type": "clinical",
  "status": "completed",
  "processing_time": 847,
  "results": {
    "processed": true,
    "confidence": 0.968,
    "output": {
      "assessment": "Hypertension detected",
      "recommendations": [
        "Monitor BP daily",
        "Low sodium diet"
      ],
      "icd_codes": ["I10"],
      "risk_level": "medium"
    }
  },
  "audit": {
    "transaction_id": "tx_789456",
    "timestamp": "2024-01-20T14:30:00Z",
    "hipaa_compliant": true
  }
}

Rate Limits: 100 requests/minute per IP in demo mode, 1000 requests/minute with API key

    "code": "AGENT_NOT_FOUND",

    "details": {

Common Error Codes

HTTP Status Error Code Description
400 INVALID_REQUEST Malformed request body or parameters
401 UNAUTHORIZED Missing or invalid authentication
403 FORBIDDEN Insufficient permissions
404 AGENT_NOT_FOUND Requested agent doesn't exist
429 RATE_LIMIT_EXCEEDED Too many requests
500 INTERNAL_ERROR Server error (will be logged)
503 SERVICE_UNAVAILABLE Temporary service outage
